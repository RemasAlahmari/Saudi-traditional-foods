🇸🇦 Saudi Traditional Foods
A web app for browsing and saving authentic Saudi recipes.
🌐 Live Demo: 

Features

Search Saudi dishes from TheMealDB API
Save favorites to database
Register & Login
Meal details with ingredients and instructions
Responsive design

Tech Stack

Frontend: React + Vite
Backend: PHP
Database: MySQL / MariaDB
API: TheMealDB


Setup
1. Clone
bashgit clone https://github.com/YOUR_USERNAME/Saudi-traditional-foods.git
2. Database

Open phpMyAdmin → create database saudi_foods_db
Import backend/database.sql

3. Backend

Copy backend/ to C:\xampp\htdocs\saudi-foods-backend\
Start Apache in XAMPP

4. Frontend
bashcd frontend
npm install
npm run dev
Open http://localhost:5173
