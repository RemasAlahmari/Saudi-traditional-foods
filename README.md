#🇸🇦 Saudi Traditional Foods
A web app for browsing and saving authentic Saudi recipes.
_🌐 Live Demo: https://saudi-traditional-foods-git-updatebranch-zad-s-projects.vercel.app/_

**##Features**


*Search Saudi dishes from TheMealDB API
*Save favorites to database
*Register & Login
*Meal details with ingredients and instructions
*Responsive design

**##Tech Stack
**
**Frontend:** React + Vite
**Backend:** PHP
**Database:** MySQL / phpMyAdmin
**API:** TheMealDB

**##Setup
**
###1. Clone
`
bash  _git clone https://github.com/RemasAlahmari/Saudi-traditional-foods.git_`

###2. Database

Open phpMyAdmin → create database saudi_foods_db

Import backend/database.sql

###3. Backend

Copy backend/ to C:\xampp\htdocs\backend\

Start Apache in XAMPP

###4. Frontend
>[!NOTE]
>You Need to enter the frontend folder because the SRC folder is inside it

`bash _cd frontend_

npm _install_

_npm run dev_`

Open http://localhost:5173
